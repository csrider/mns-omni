The hardware wizard is the preferred way to define a new Omni appliance!
Navigate to "Other Screens", Hardware Sign Database, then click on the wizard button.

  • Select Omni (Evolution)
  • Select Omni Appliance, then Next
  • Enter the Device ID
  • Enter the MAC Address
  • Enter the IP Address (only if using static address)
  • Enter the Device Username (usually numerical and same as SIP number)
  • Enter the Device Password (usually numerical and same as SIP number)
  • Select the Timezone
  • Enter the SIP Extension/Number (should correspond with Extension Manager entry)
  • Enter a description (MAC address will automatically be appended when done, for convenience)
Click Next and then click "Yes, Save New Device". Then the server will add the Omni device, provided there are licenses available.

NOTE: If using the hardware, a corresponding Extension-Manager record will also be automatically created for you. If you do not use the wizard, you will need to manually create an Extension-Manager entry, yourself. If possible, it's suggested that you use an existing Omni definition as a template (modify it and save-as-new). SM_INSERT_BODY_END
